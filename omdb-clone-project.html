OMDB API

Open Movie Database

The OMDB API is RESTFULL web service to obtain movie information All content and images on the site are contributed and maintained by our users

If you find the service useful, please consider making one time donation, or become a patron.

Poster API

The poster api is only available to patrons.

Currently, over 280,000 posters updated daily with resolution up to 2000x3000.

Usage

Send all data request to:

http://www.omdbapi.com/?apikey=[yourkey]&

Poster API requests:

http://img.omdbapi.com/?apikey=[yourkey]&

Parameters

By ID or Title

Parameter Required Valid Options Default Value Description
i <empty> A valid IMDb ID (e.g. tt1285016)
t <empty> Movie title to search for.
type movie, series, episode <empty> Type of result to return.
y <empty> Year of release.
plot short, full short Return short or full plot.
r json, xml json The data type to return.
callback <empty> JSONP callback name.
v 1 API version (reserved for future use).

*Please note while both "i" and "t" are optional at least one argument is required.

Parameter Required Valid Options Default Value Description
yes <empty> <empty> Movie title to search for.
type movie, series, episode <empty> Type of result to return.
y <empty> Year of release.
r json, xml json The data type to return.
page 1-100 1 page number to return.
callback <empty> JSONP callback name.
v <empty> 1 API version (reserved for future use).
